Why Learn Bootstrap?

Bootstrap is the world's most popular front-end open source toolkit for building responsive, mobile-first sites. With Bootstrap, you can:

  • Build responsive layouts quickly with its grid system
  • Use pre-styled components (buttons, cards, navbars, etc.)
  • Customize everything with Sass variables and mixins
  • Ensure consistency across browsers and devices

Lesson 1: Bootstrap Grid System

Key Concept

Bootstrap's grid system uses containers, rows, and columns to layout content. It's built with flexbox and is fully responsive.

How the Grid Works

Bootstrap's grid system divides the screen into 12 columns. You specify how many columns an element should span at different breakpoints.

HTML Structure:
<div class="container">
  <div class="row">
    <div class="col-md-8">Content (8 columns on medium+)</div>
    <div class="col-md-4">Sidebar (4 columns on medium+)</div>
  </div>
</div>

Grid Breakpoints

Bootstrap has 6 responsive breakpoints:

  • xs (default): <576px
  • sm: ≥576px
  • md: ≥768px
  • lg: ≥992px
  • xl: ≥1200px
  • xxl: ≥1400px

Lesson 3: Customizing Bootstrap

Key Concept

Bootstrap is designed to be customizable. You can override its styles, use CSS variables, or modify the Sass source files.

Customization Methods

Override Bootstrap styles with your own CSS (order matters!):

Custom CSS File (load after Bootstrap):
/* Override primary color */
.btn-primary {
    background-color: #6f42c1;
    border-color: #6f42c1;
}

/* Override navbar background */
.navbar-light {
    background-color: #f8f9fa !important;
    border-bottom: 2px solid #0d6efd;
}

Bootstrap 5 includes CSS custom properties (variables) for easy theming:

Using CSS Variables:
:root {
    --bs-primary: #6f42c1;
    --bs-primary-rgb: 111, 66, 193;
    --bs-body-bg: #f8f9fa;
    --bs-body-color: #212529;
    --bs-border-radius: 0.5rem;
}

For full control, customize Bootstrap's Sass variables before compiling:

Custom SCSS File:
// Override default variables
$primary: #6f42c1;
$enable-rounded: false;
$font-size-base: 1.1rem;

// Import Bootstrap
@import "~bootstrap/scss/bootstrap";

Lesson 4: Bootstrap Utility Classes

Key Concept

Bootstrap includes hundreds of utility classes for quickly styling elements without writing custom CSS.

Common Utility Classes

Spacing Utilities

Use m for margin and p for padding, followed by direction and size.

Padding: 3 (p-3)
Margin: 3 (m-3)
Margin-top & padding-top: 4 (mt-4, pt-4)
Spacing Examples:
<div class="mt-3">Margin top 1rem</div>
<div class="p-4">Padding 1.5rem</div>
<div class="mx-auto">Center horizontally</div>
